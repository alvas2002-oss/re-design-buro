# RE Design Buro — Design System & Slides

Дизайн-система и материалы для презентаций студии **RE Design Buro** (rmexp.ru) — дизайн и ремонт квартир в Москве под ключ.

---

## Файлы

| Файл | Что внутри |
|------|-----------|
| `re-design-system.md` | Дизайн-система: цвета, шрифты, типографика, паттерны — для вставки в Claude |
| `re-slides-tz.md` | ТЗ на 2 слайда + точные тексты — для вставки в Claude |
| `tz-claude-design.md` | Расширенное ТЗ с ASCII-схемами макетов — для Claude Design |
| `rmexp-design-system.html` | Визуальный стайлгайд — открыть в браузере |
| `rmexp-slides.pptx` | Готовые слайды в PowerPoint |

---

## Как использовать в Claude

1. Открой новый чат на claude.ai
2. Вставь содержимое `re-design-system.md`
3. Затем вставь содержимое `re-slides-tz.md`
4. Напиши: *«Сделай слайды по ТЗ, текст не менять»*

## Дизайн-система — быстрая справка

**Цвета:**
- `#1A1714` — основной тёмный
- `#B8965A` — золотой акцент
- `#FAF6EF` — светло-песочный
- `#F8F7F5` — тёплый белый фон

**Шрифты:**
- Cormorant Garamond — заголовки и цены
- Inter — всё остальное

**Логотип:** RE. (точка всегда золотая)
